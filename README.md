# 🚀 03Guri Controle

Aplicativo Android desenvolvido para auxiliar no controle de estoque e registro de vendas de uma pequena empresa do setor de tecnologia.

---

## 📱 Sobre o Projeto

O **03Guri Controle** foi criado com o objetivo de substituir processos manuais (cadernos e planilhas) por uma solução digital simples, prática e eficiente.

O aplicativo permite que colaboradores realizem o controle de produtos e registrem vendas de forma rápida, reduzindo erros e melhorando a organização das informações.


## ✨ Funcionalidades

* 🔐 Tela de Login (acesso simples)
* 📦 Controle de Estoque

  * Listagem de produtos
  * Visualização de quantidade disponível
  * Preços atualizados
* 💸 Registro de Vendas

  * Seleção de produtos
  * Controle de quantidade
  * Atualização automática do estoque
* 📊 Cálculo automático do valor total


## 🧠 Tecnologias Utilizadas

* **Kotlin**
* **Android Studio**
* **XML (Layouts)**
* **RecyclerView**
* **Material Design**
* **Arquitetura simples (Activities + Adapter)**

## ⚙️ Como Executar o Projeto

1. Clone o repositório:

```
git clone https://github.com/seu-usuario/03guri-controle-app
```

2. Abra o projeto no Android Studio

3. Configure o SDK no arquivo `local.properties`:

```
sdk.dir=C:\\Users\\SEU_USUARIO\\AppData\\Local\\Android\\Sdk
```

4. Execute o projeto em um emulador ou dispositivo físico
